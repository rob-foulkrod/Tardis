# 🚀 Tardis

> **"Bigger on the inside. Faster than time. Ready for anything."**

Welcome to **Tardis**, the web project that dares to dream beyond the ordinary. Inspired by the legendary time machine, this codebase is your gateway to building web experiences that transcend the boundaries of space, time, and imagination.

## 🌌 What is Tardis?
Tardis is an aspirational .NET web application, designed to be:
- **Scalable**: Like the TARDIS itself, our architecture is bigger on the inside.
- **Fast**: Blink and you'll miss it—performance is at the heart of every line.
- **Flexible**: Whether you're building for today or tomorrow, Tardis adapts.
- **Fun**: Code should be an adventure, not a chore.

## ✨ Features
- Modern ASP.NET Core MVC structure
- Razor views for beautiful UI
- Bootstrap-powered styling
- Ready-to-go authentication & error handling
- Easy to extend, easy to deploy

## 🛠️ Getting Started
1. **Clone the repo**
   ```pwsh
   git clone https://github.com/rob-foulkrod/Tardis.git
   ```
2. **Run the app**
   ```pwsh
   cd Tardis/src/Tardis.Web
   dotnet run
   ```
3. **Open in browser**
   Navigate to `https://localhost:5001` and step inside your very own Tardis.
